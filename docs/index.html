Core Features

A novel hybrid framework combining evolutionary algorithms with physics-informed fitness.

01

Curriculum Learning

GenePhase stages: SEED → SPROUT → GROW → BLOOM. Gradual complexification inspired by NEAT.

02

Neural Memory

LSTM/GRU maintains hidden state across runs. Learn from searches, predict trajectories.

03

Multi-Objective

NSGA-II Pareto optimization. Trade-offs between spine coupling, ear uniformity, structure.

04

Physics-Informed

FEM modal analysis via JAX autodiff. ABH detection. Zone-specific frequency response.

05

CNC Export

Direct STL and G-code generation. Topology-optimized cutouts with Bézier contours.

06

Research-Backed

70+ distilled papers: lutherie, body resonance, vibroacoustic therapy, plate physics.
